Data set: GTEx RNA expression, which contains RNA-seq profiles of 8555 samples taken in different tissue types. For a visual summary, please refer to GTEx summary page.
Processing method: The data is processed with TOIL and made available by the Xena Project: sequencing reads are mapped to genes using RSEM, and transcript abundance is quantified in TPM. The data provided on XenaHub were log-transformed by the formula log_2(TPM + 0.001). In this visualization, the expression is transformed to log10 scale by default. Although linear scale is also supported, it is not usually very helpful with RNA expression data.
